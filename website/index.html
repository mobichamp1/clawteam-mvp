Why ClawTeam?

Agents Spawn Agents

Leader agents spawn specialized workers with one CLI command. Each gets its own git worktree and tmux window automatically.

Real Inter-Agent Messaging

Workers report status, send results, and coordinate via a file-based inbox. No hard-coded orchestration logic.

Task Dependencies

Kanban board with pending → in_progress → completed. Tasks auto-unblock when dependencies finish.

Any CLI Agent

Works with Claude Code, Codex, OpenClaw, nanobot, and any agent that runs from the command line.

Git Worktree Isolation

Each agent works on its own git branch. No merge conflicts, real diffs, real checkpoints.

One-Command Templates

TOML templates define team archetypes. clawteam launch hedge-fund spins up 7 analyst agents instantly.
